Useful Commands!

terminal

  • Ctrl + Alt + T
    launch standalone terminal
  • Tab
    auto-complete rest of directory/filename
  • Tab Tab
    show what auto-complete options are available
  • cd
    (no argument) goes to home directory

VScode

  • Line number
    select the entire line just by clicking the line number!
  • Alt + Click
    make changes on multiple "alt + clicked" lines
  • Alt + ↑ or ↓
    move all highlighted lines as a block
  • Ctrl + /
    turn all highlighted lines into a // comment
  • Ctrl + D
    select the word at the cursor
  • Ctrl + Shift + L
    Select all occurrences (same line/word matches) of current selection
  • F2
    refactor variable name at text cursor (rename every time it is referenced)
  • Copy and Paste (automatic)
    - copy line at cursor automatically with Ctrl + C without selecting it
    - Ctrl + V paste the copied line to the next line automatically without pressing enter
  • EXTRA: Custom Command
    Ctrl + Shift + P opens command search. Settings icon customizes any command

git terminal

  • git remote -v
    show the remote push and pull URLs
  • git log
    show the last git commit messages
  • git restore .
    roll back project to last git commit
